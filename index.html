Key Technology Insights

⚙ Custom Theme — Flexibility vs Missing Features

  • Full design control — but missing standard conversion features that modern Shopify themes include out-of-the-box
  • Missing natively: Image zoom, product tabs, sticky ATC, Quick View, size guide, breadcrumbs (hidden via CSS), collection banners
  • Impact: Every missing feature must be custom-coded or added via apps — increasing JS payload, maintenance cost, and potential for conflicts
  • Consider: Migrating to a modern theme (e.g., Motion, Be Yours) could deliver 10+ missing features out-of-the-box, reducing app dependency and page load

⚠ GoKwik Integration — Urgent Debugging Needed

  • GoKwik controls the entire purchase flow — cart drawer, /cart redirect, and checkout
  • "checkout element not found" fires 8+ times per page load on every page — the script is looking for DOM elements that don't exist in the custom theme
  • "gokwikCheckout is not defined" — the checkout function isn't loading into the global scope properly
  • Risk: If these errors affect even 2-3% of checkout attempts, the revenue impact at Deyga's traffic volume is significant
  • Action: Share console logs with GoKwik support, verify element selectors match theme DOM, test checkout end-to-end

📊 Analytics — Good Foundation, Verification Needed

  • Detected: GTM container, GA4, Facebook Pixel, Google Ads tag, Pinterest Tag, Microsoft Clarity
  • Unknown: Whether GA4 ecommerce events (view_item, add_to_cart, begin_checkout, purchase) are firing correctly — requires GA4 Viewer access to confirm
  • Why it matters: Without confirmed ecommerce tracking, every marketing spend decision is based on incomplete data — you can't optimize what you can't measure
  • Next step: Grant GA4 Viewer access so we can verify event tracking and provide a full conversion funnel analysis

🔍 SEO — Structural Gaps Limiting Organic Visibility

  • 3 H1 tags on homepage ("Our Best Sellers", "Shop By Concern", "Our Ingredients") — Google expects exactly 1 per page
